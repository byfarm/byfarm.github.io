Roll Grind Optimization

Source Code

Summary

This project was my Senior Design Project for Gonzaga University. Kaiser Aluminum is trying to better predict when their aluminum rolls are going to fail based on data collected from their PLC's. My group's solution was to create a MySQL database to efficiently store the data we needed, and created a web frontend to display our calculations.
