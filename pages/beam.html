Beam Analyzer

Source Code

Summary

This program parsed and decoded a text file generated by user input to understand the loading conditions on a beam. It then interpreted these loading conditions and calculated the shear and bending moment at each point along the beam. Data analysis was then performed on the results to find the maximum shear and bending moment.

Improvements to Make

Having the program calculate the deflection across the beam would be a great, however then information about the dimentions and material of the properties would have to be known. A custom numerical integrator would also need to be created to be compatible with the input language.
